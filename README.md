Simple PHP File Manager

Supports uploading, downloading and removing files, creating and removing folders, searching by name, sorting by name, size, type and date of last modify, ascending and descending ordering.
Shows files and folders in the table.

Icons are customizable by saving them in format extension.png or by group type (like document.png, music.png), which apply selected icon for every extension in group.
Supported groups:
  Music - mp3, aac, wav, flac, ogg, m4a, aiff
  Video - mp4, avi, mkv, mov, wmv, webm, mpeg, mpg, m4v
  Image - jpg, jpeg, png, gif, bmp, webp, tiff, svg
  Document - pdf, doc, docx, xls, xlsx, ppt, pptx, txt, md, rtf, csv
  Archive - zip, rar, 7z, tar, gz, bz2, xz
If the icon is not avaiable, unknown.png will be used.

You can enable or disable uploading, downloading and creating folders. You can modify max. upload limit.
